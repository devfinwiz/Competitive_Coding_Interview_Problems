3. **Open LeetSync extension**: üåê Open the LeetSync extension you just added in step 2.

4. **Complete Setup**: ‚úîÔ∏è Click on 'Complete Setup', then 'Login with GitHub', and finally 'Authorize' to authenticate LeetSync with your GitHub account.

5. **Re-open LeetSync extension**: üîÑ Re-open the LeetSync extension window.

6. **Copy Repository URL**: üìã Copy the URL of the forked repository (the one visible on your profile) and paste it into the LeetSync extension window.

10. **Raise Pull Request**: üéâ Click on 'Contribute' from the forked repository and raise a pull request with the LeetCode problem number as the title of the pull request.
